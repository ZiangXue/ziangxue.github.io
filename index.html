The cloud diary: The project will mainly focus on training convolutional neural networks (CNNs) for the task of classifying images of different types of clouds. The desired data set will be photos of clouds taken from ground-based camera, e.g. your smart phone. The trained CNN will be deployed on a (possibly web based) application on which users can upload their pictures and get a classification of the cloud contained the picture. Future development of the application can be associating the pictures with location information. This would help to make local weather forecast (especially rain forecast) more accurate. Project Goals: - Create a dataset of the aforementioned photos of cloud to train the CNN with. - Find ways to isolate noise from cloud with pre-processing - Train the CNN to categorize picture between 3 main types of clouds - Build an application centered by the trained CNN. - Perhaps, find a way to combine this forecast with location data and compare to local rain forecast.
